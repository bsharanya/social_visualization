We used python, d3 and jquery for the entire project.

In order to run this project you need the python framework setup.

Python version: python 2.7
Install the following packages using "pip" package manager:
1. pip install flask
2. pip install simplejson
3. pip install requests
4. pip install numpy
5. pip install scipy
6. pip install operator
7. pip install subprocess
8. pip install sys

Once you are done installing the above packages run the following steps:
1. cd social-coding
2. python run.py
3. open http://localhost:5000 in Google Chrome
